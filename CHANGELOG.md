# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-01

### Added

- Initial release of the Islamic Open Finance developer tools
- CLI tool (`iof`) for interacting with the IOF platform from the terminal
- Login and authentication commands with OAuth 2.0 device flow support
- API testing utilities for validating endpoint responses against the OpenAPI spec
- Contract validation commands for verifying Islamic finance contract schemas
- Shariah compliance validation to check contracts against AAOIFI standards
